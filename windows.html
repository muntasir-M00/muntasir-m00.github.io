About Windows

Windows is Microsoft's powerful and popular operating system. The two most used modern versions today are:

  • Windows 11 — the newest version with modern design, better performance, and enhanced security (2025 Update - 25H2)
  • Windows 10 — still very popular, stable, widely supported, and officially supported until October 2025 (with extended support options available after)

System Requirements

Windows 11

  • 64-bit processor (1 GHz or faster, 2+ cores)
  • 4 GB RAM (8 GB+ recommended)
  • 64 GB storage
  • DirectX 12 compatible graphics with WDDM 2.0 driver
  • TPM 2.0 + Secure Boot capable
  • 720p display > 9 inches

Windows 10

  • 1 GHz or faster processor
  • 1 GB RAM (32-bit) or 2 GB RAM (64-bit)
  • 16 GB (32-bit) or 20 GB (64-bit) storage
  • DirectX 9 graphics with WDDM 1.0 driver
  • 800×600 display

How to Download & Install Windows (Step by Step)

  1. Check if your computer meets the requirements for Windows 10 or 11.
  2. Back up all important files and data.
  3. Choose which version you want (10 or 11).
  4. Visit the official Microsoft link above for your chosen version.
  5. Select one of these options:
    • Upgrade this PC now (Installation Assistant)
    • Create installation media (USB or ISO file)
  6. If you chose ISO:
    • Download the ISO file
    • Use Rufus or Media Creation Tool to make a bootable USB
  7. Boot from the USB (restart PC → press boot menu key → select USB)
  8. Follow the on-screen Windows setup instructions.

How to Make Bootable USB from ISO (Using Rufus)

  1. Download Rufus → https://rufus.ie
  2. Insert a USB drive (minimum 8 GB)
  3. Open Rufus
  4. Select your USB drive
  5. Click "SELECT" and choose your Windows ISO
  6. Leave default settings (or choose GPT/UEFI for modern PCs)
  7. Click START → wait until finished
Important Warnings:
  • All data on the USB will be deleted!
  • Use only official Microsoft ISO files
  • Do not remove USB during creation
  • Windows 11 requires TPM 2.0 & Secure Boot (Rufus can bypass if needed)
